## Server specifications  

PHP 7.4^  
MySQL database  
composer 2.0  

## Installation  
1.- Needs create database in your server  
- Example::  
- run > mysql -u root -p  
- run > create database database_name;  
run > composer install  

2.- run > php artisan migrate
